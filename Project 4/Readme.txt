N queens Steps:
1. Create a list of N dummy variables
2. Search for a configuration of N queens
3. Generate all possible positions for queen Q
4. Pick one of such position
5. Check whether the queen Q is safe(not on the same row) & (diagonal distance)
6. If queen is safe so that means given configuration is valid configuration.
